## 문서 1 — 자기소개서 (하이브리드, **600자 이내**)

리드(정체성, 1~2문장) + 소제목 3개. 각 소제목 2문장 내외로 압축.

- **리드**: 정보·서비스에 약한 사용자가 쉽게 이해하도록, 데이터에서 출발해 화면까지 직접 설계하는 기획자. AI는 도구로 쓰되 판단은 직접 책임.
- **① 지원동기**: 베베루시(아기)·루시케어(마음 약한 사람) = 취약 사용자 배려가 핵심. ALLAW에서 "법을 모르는 사람도 이해하게" 설계한 경험과 같은 결. 약자를 위한 설계가 가장 잘하고 싶은 일.
- **② 직무역량**: ALLAW 89명 분석 → UX 전략 검증·기각 → KCI 등재 (데이터 기반). Clog 1인 100% 화면설계 (실행·Figma). Second Brain·EEL 콘텐츠 분업 (AI 활용).
- **③ 입사 후 각오**: 빠른 학습 + 약자 배려 서비스를 데이터로 키우고 싶다. (경력 약점 언급 X)

제약: 공백 포함 600자 이내. 소제목은 짧게(라벨 수준).

## 문서 2 — 경력기술서 (프로젝트 중심)

상단 한 줄: "기획부터 화면 설계·검증까지 직접 끌고 간 프로젝트를 중심으로 정리합니다." (약점 언급 X)

프로젝트 블록 (각: 기간 / 역할 / 담당업무 / 핵심성과 / 사용도구):

1. **ALLAW — MS컴퍼니 재직** (2022.06–2023.01): 초기 UI/UX 기획·화면설계·스토리보드, 직군 통역(의사결정 1장 정리) → 초기창업패키지 7,000만원 선정. 툴: Figma, React Native(웹앱).
2. **ALLAW 사용자 연구 — 석사** (2024–2025): 설문 89명 + 사용성 5명 → UX 전략 3개 검증, 1개 기각·재설계 → KCI 등재 논문(JNCIST 2025). 툴: 설문·통계, Figma.
3. **Clog — 클라이밍 기록 앱** (2024–2025): 1인 100% 기획·리서치·UI·프로토타입, 12화면 설계문서. 툴: Figma.
4. **EEL — 레진 가구 브랜드** (2025–2026): 사업기획 + 웹사이트 설계 + AI 콘텐츠 제작(기준 수립→AI 시안 양산→선별). 툴: 웹, AI 도구.
5. **해외 업무 — 애술린/오로플러스** (2023–2026, *짧게*): 해외 거래처 영어 소통, 도서 수입·수출, B2B 대량 판매 제안. (직무 외, 영어·성실성 근거)

ALLAW를 재직(1)+연구(2)로 분리해 "ALLAW를 수년간 깊게 판" 인상.
