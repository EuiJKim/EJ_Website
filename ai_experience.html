저는 AI를 일회성 도구로 쓰는 사람이 아니라, 창작과 사업 운영 전반에 AI를 통합해 한 사람이 처리할 수 있는 범위를 확장해온 사람입니다. 디자인을 만들고(Build), 사업을 굴리고(Operate), 광고로 성장시키는(Grow) 사이클 전체에서 AI를 협업 파트너로 사용합니다.

PILLAR 01

BUILD · 만들기

디자인 · 코드 · 시스템을 산출물까지

포트폴리오 웹사이트 + 다국어 i18n 시스템

메인 + 3개 케이스 스터디 + 회사별 지원 페이지까지 5개 이상의 라우트를 Claude Code와 협업해 직접 구축. ?lang=en URL 파라미터 기반 영문 강제 진입, localStorage 동기화 i18n 시스템 설계.

디자인 시스템 + 자동화 파이프라인

컬러·타이포·컴포넌트 토큰을 단일 CSS로 운영, Figma 결정이 코드에 바로 반영되도록 디자이너 1인이 디자인-개발 동시 운영. 이력서·자기소개서·포트폴리오 PDF를 HTML 한 번 수정으로 동시 재생성하는 파이프라인(Puppeteer + pypdf) 구축.

학술 분석 협업 (ALLAW · KCI 게재)

89명 사용자 응답 분석에서 코드 생성·차트 시각화·발화 클러스터링· 논문 초안 구조화를 AI와 반복 협업. 판단은 사람이, 반복은 AI가 가져가는 흐름으로 학술 게재 수준까지 완성.

PILLAR 02

OPERATE · 운영

실제 매출이 발생하는 e-커머스 운영

EEL — 레진 가구 e-커머스 사업

실제 제품을 판매하는 실서비스(eel-studio.me). 웹사이트 디자인·BTO 주문 플로우·3D 커스터마이저·결제 연동까지 1인이 기획·운영. 포트폴리오 프로젝트가 아니라 매출이 발생하는 사업의 일부.

상품 콘텐츠 자동화

신규 제품·시즌 콘텐츠 제작 시 AI로 상세페이지 카피 초안, 이미지 보정, 3D 렌더 컨셉, 제품 소개 영상 스크립트를 동시 진행. 한 제품의 콘텐츠 사이클을 1주일 → 1일로 단축.

CS · 운영 표준화

FAQ 정리, 주문 응대 템플릿, 배송 안내 문구를 AI 자가검토 루프 로 표준화. 운영자 1인 체제에서 응대 일관성을 유지하고, 반복 응대 시간을 의사결정 시간으로 재투자.

PILLAR 03

GROW · 성장

마케팅 · 광고 · 데이터 이터레이션

Instagram 광고 운영 (AI 카피·이미지·타겟팅)

EEL 사업의 인스타그램 광고 카피·후킹 라인·이미지 컨셉을 AI로 다수 변주 생성해 A/B 테스트. 사람 1인이 캠페인 카피 5개 동시 운영. 타겟팅 가설을 빠르게 세우고 데이터로 폐기/강화.

콘텐츠 캘린더 + 인사이트 합성

주간 콘텐츠 캘린더, 트렌드 키워드 정리, 경쟁 브랜드 모니터링을 AI에게 1차 작업 위임. 사람은 톤·인사이트·실행 우선순위 결정에만 집중.

광고 성과 분석 → 다음 가설

지난 주 광고 성과 → 가설 기각/강화 → 다음 주 운영안을 한 페이지 Brief로 자동 정리. AI솔루션기획에서 요구되는 "데이터→가설→실행" 사이클을 광고 단위에서 직접 운영.

CORE PRINCIPLE

AI가 판단을 대신하지 않는다. 사람이 판단에만 집중할 수 있도록 주변 마찰을 최대한 줄인다. 한 사람이 디자인·운영·마케팅을 동시에 끌고 갈 수 있는 이유는, AI가 반복을 가져가고 사람이 결정을 가져가는 흐름이 일상화되어 있기 때문이다.

3-Step Workflow: ① AI가 3~5개 가설·시안을 먼저 만든다 → 사람이 고른다 · ② 산출물의 80%를 AI가 채우고 → 사람이 검토한다 · ③ AI에게 자가 점검을 시켜 누락을 0%로 줄인다.
